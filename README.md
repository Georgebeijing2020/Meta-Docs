# Meta-Docs

## 使用方法
### 设置源
```shell
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
```
### 安装
```shell
pip install mkdocs-material
```
### 预览修改
```shell
mkdocs serve
```
### 参考文档

https://squidfunk.github.io/mkdocs-material/setup/
